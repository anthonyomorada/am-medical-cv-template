# Contributing to Academic Medicine Portfolio Template

Thank you for your interest in improving this template! This project helps medical professionals create professional online portfolios.

## üéØ Ways to Contribute

### 1. Report Issues
Found a bug or problem?
- Check [existing issues](https://github.com/anthonyomorada/am-medical-cv-template/issues) first
- Open a new issue with clear description and steps to reproduce
- Include screenshots if relevant

### 2. Suggest Features  
Have an idea for improvement?
- Open a [discussion](https://github.com/anthonyomorada/am-medical-cv-template/discussions)
- Explain the use case and how it helps medical professionals
- Consider if it maintains simplicity for non-technical users

### 3. Share Your Site
Using this template?
- Post in [Show and Tell](https://github.com/anthonyomorada/am-medical-cv-template/discussions)
- Help others by sharing what worked well
- Get featured in the README examples section

### 4. Improve Documentation
- Fix typos or unclear instructions
- Add troubleshooting solutions
- Create tutorial videos or guides
- Translate documentation (if multilingual)

## üö´ What We Don't Accept

- Features requiring JavaScript knowledge from users
- Complex customizations that break simplicity
- Features only useful for non-medical use cases  
- Changes that require paid services

## üìã Design Principles

This template prioritizes:
1. **Zero coding required** - Must work for non-technical users
2. **Markdown only** - No HTML/CSS editing needed for basic use
3. **Mobile-first** - Program directors view on phones
4. **Free forever** - GitHub Pages hosting only
5. **Academic professional** - Conservative, credible design

## üí¨ Code of Conduct

- Be respectful and professional
- Help junior colleagues learn
- Assume good intentions
- Focus on helping medical professionals

1. Fork the repository

3. Test changes thoroughly

5. Submit PR with clear description

# Publications

This folder is for storing PDFs of your published research papers and articles.

## How to Use

1. Upload your publication PDFs to this folder
2. Name them descriptively (e.g., `2023-smith-diabetes-outcomes.pdf`)
3. Link them in your `cv.md` file:

```markdown
**Your Name**, Co-authors. Article title. *Journal Name*. 2023;10(2):123-456. PMID: 12345678. [[PDF](./publications/2023-smith-diabetes-outcomes.pdf)] [[PubMed](https://pubmed.ncbi.nlm.nih.gov/12345678/)]
```

## Tips

- Use descriptive filenames with year, first author, and topic
- Keep file sizes reasonable (compress if needed)
- Only upload papers you have rights to distribute
- Consider adding preprints from bioRxiv, medRxiv, etc.

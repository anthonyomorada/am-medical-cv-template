# Presentations

This folder is for storing PDFs of your conference posters, presentations, and slides.

## How to Use

1. Upload your presentation PDFs to this folder
2. Name them descriptively (e.g., `2023-asco-poster-lung-cancer.pdf`)
3. Link them in your `cv.md` file:

## Tips

- Export posters as PDF for best compatibility
- Compress large files to reduce loading time
- Include year and conference name in filename
- Works great for posters, slides, and abstracts
